# andone 기획 방향

todo를 완료하면 나만의 캐릭터(문어)를 키우는 앱.
RPG 전투 방식이 아닌 캐릭터 상태 관리 방식으로 운영.

## 핵심 루프
```
todo 완료 → 골드 획득 + 기분 수치 상승 → 골드로 꾸미기 아이템 구매 → 캐릭터/배경 꾸미기
```

## 캐릭터 시스템
- 캐릭터: 문어 픽셀 아트 (`assets/image/character/my_pet_right.png`, `my_pet_left.png`)
- 상태 수치: 기분 (0~100) → 나중에 배고픔/피로도 추가 예정
- 수치에 따라 캐릭터 표정/스프라이트 변경 예정
- 좌우 유영 애니메이션 적용 (방향에 따라 스프라이트 전환)

## 골드 시스템
- todo 완료 시 난이도에 따라 골드 지급 (난이도 1→10골드 ~ 5→50골드)
- 골드로 악세서리, 배경 소품 구매 예정

## 꾸미기 시스템 (예정)
- 악세서리: 모자, 안경, 스카프 등 캐릭터에 직접 착용
- 배경 소품: 화분, 책상, 조명 등 방 꾸미기
- 장비 탭 → 상점/꾸미기 탭으로 전환 예정

## Todo 시스템
- 반복 설정: 없음 / 매일 / 매주 (요일 선택)
- 반복 없음: `isCompleted` 사용, 해당 날짜에만 표시
- 반복 있음: `lastCompletedDate` 사용, 매일 자동 초기화
- 시작일 기준으로 반복 시작
- 반복 있는 todo는 항상 메인 화면에 표시 (수정/삭제 접근 가능하도록)
